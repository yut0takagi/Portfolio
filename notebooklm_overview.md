# Yuto Takagi – Portfolio: NotebookLM Overview

本ファイルは、NotebookLM に取り込むことを想定した概要資料です。サイトの目的、構成、主要セクション、スキル、プロジェクト一覧、タイムライン、および音声読み上げ用の原稿案を含みます。

## サイト概要
- 目的: 機械学習と Web アプリ開発を中心とした活動・成果の紹介と相談窓口。
- 対象読者: 採用担当・共同開発者・クライアント。
- スタック: React (CRA), Tailwind CSS, Framer Motion, Vanta.js。
- 特徴: スクロール連動の動的演出（パララックス、ピン留めショーケース）、アクセシビリティ配慮、OGP/Twitter カード、構造化データ。

## サイト構成
- Hero: 左に写真、右に自己紹介と CTA（Projects/Contact/音声再生）。
- Highlights: プロジェクト数などのハイライト指標。
- Story (Scroll Showcase): スクロールで代表作が順に切り替わる演出。
- Featured Projects: 代表プロジェクト抜粋。
- Skills: 言語/フレームワーク/ML・データ/ツールの一覧。
- Timeline: 大学入学から現在までの軌跡。
- Projects: すべてのプロジェクト一覧（画像・説明・技術タグ・外部リンク）。
- About: 自己紹介（日本語）。
- Contact: 連絡手段（メール/ GitHub）。

## スキル（抜粋）
- Languages: Python, JavaScript, TypeScript, SQL
- Frameworks: React, Tailwind CSS, Streamlit, Flask
- ML / Data: Pandas, scikit-learn, NumPy, Matplotlib
- Cloud / Tools: Firebase, GitHub Actions, Docker (basic), Vercel/Render
