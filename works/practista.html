Practista for All

教育支援サービスとして学生,塾関係者,学校関係者,保護者へ包括的なサポートを行います。
従来よりも、円滑な情報共有,学習に集中できるサービスです。

🎓 学生向け学習支援アプリ - Practista for Student

使用技術：Flask, SQLite, RAG, ファインチューニング, Tailwind CSS

学生の勉強をAIで最適化するサービスとして開発しました。具体的な機能は以下のとおりです。

  • オンライン教材管理システム
  • 課題提出システム
  • 課題解決用チャットボットシステム
  • 問題作成システム
  • 進路相談用チャットボットシステム

オンライン教材管理システム

従来のオンライン教材は、各サービス内で既習範囲を管理しています。
そのため、利用者が複数のサービスを並行して利用する場合は、相互に確認する必要があり、面倒であると言えます。
さらに、「ICT教育」が進んでいることで、オンライン指導が増加すると予想でき、生徒は目に見えない教材を膨大に抱え 活用しきれない事象が発生するでしょう。 そこで、本サービスでは、各オンライン配信サービスでの学習事項を取得し、「生徒のカリキュラムを一元管理するシステム」を提供します。
これを利用することで、生徒の学習が見える化され、学習意欲に繋がります。

課題提出システム

学校との教材や情報共有システムを提供します。 Tailwindcssを利用したWebアプリケーションであるため、デバイスを選ばずどこからでもすぐにアクセスできUI/UXが向上しました。

課題解決用チャットボットシステム

学校教材や塾の教材、自宅学習教材などを進める中で、一人で解決できない部分が多くあるでしょう。 その中で、教材にとらわれず写真を撮影するだけで、その問題の解法について質問できます。

問題作成システム

問題集を利用した学習では、ある程度決まった問題でのアウトプットが行えるものの「身についているのか」判断するのは難しいです。 そのため、「問題集では解けるのにテストでは...」という学生が続出するのです。 そこで、実力試しに特化したテスト形式の問題が必要になります。ただし、学生一人ひとりの学力レベルや既習事項内容に遵守する必要があります。 そのため、学生一人ひとりの情報を参照したAIがパーソナライズしたテスト形式の問題を生成します。 また、その回答を記録することで定着度を数値化し学力の推移を可視化できます。

進路指導用チャットボットシステム

進路指導に特化したチャット形式のAIを提供します。 全国の高校,大学の情報を学習したAIで、学生一人ひとりの専門性や夢を汲み取り、マッチするベストな学校を提案します。
